# Security Notes

- Keine echten Secrets im Repo; `.env` Werte bleiben lokal.
- JSON Stores enthalten Demo-Daten. Produktive Stores muessen separiert und
  durch Zugriffskontrollen geschuetzt werden.
- Agent kommuniziert aktuell offline. Neue Integrationen muessen samt Scopes in
  diesem Dokument nachgetragen werden.
